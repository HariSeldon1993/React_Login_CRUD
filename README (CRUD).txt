PROJETO LOGIN - CRUD

NECESSÁRIO: 
-XAMPP
-MYSQL WORKBENCH
-VISUAL CODE (SE NÃO OPTAR PELO CMD)

1- BAIXE OS ARQUIVOS E DESCOMPACTE-OS DE FORMA QUE NA PASTA ESCOLHIDA SEJAM VISUALIZADAS AS PASTAS 'CLIENT' E 'SERVER';

2- INSTALE O XAMPP;

3- INSTALE O MYSQL WORKBENCH;

4- USANDO O CMD OU O CODE, ACESSE O CAMINHO DA PASTA 'CLIENT' E DÊ O COMANDO 'npm install'; FAÇA O MESMO COM A PASTA 'SERVER';

5- ABRA O XAMPP E INICIALIZE OS SERVIÇOS 'APACHE' E 'MYSQL'. POR PADRÃO DEIXE A PORTA DO SERVIÇO 'MYSQL' COMO '3306';

6- ABRA O MYSQL WORKBENCH E COPIE, COLE E EXECUTE OS SEGUINTES COMANDOS:

>>>
	CREATE DATABASE PWBE;

	USE PWBE;

	CREATE TABLE usuarios(
	idusuarios int auto_increment not null primary key,
	email varchar(50) not null,
	password varchar(200) not null
	);

	CREATE TABLE cadastro (
	id 	INT auto_increment NOT NULL PRIMARY KEY,
	nome VARCHAR(45) NOT NULL,
	idade INT NOT NULL,
	cpf VARCHAR(11) NOT NULL,
	matricula VARCHAR(10) NOT NULL,
	curso VARCHAR(50) NOT NULL
	);
<<<

7- NO ARQUIVO INDEX.JS DA PASTA 'SERVER' VERIFIQUE SE O NOME DA DATABASE É O MESMO DA DATABASE CRIADA NO MYSQL. FAÇA O MESMO PARA A PORTA E A SENHA NO ARQUIVO. REPITA A VERIFICAÇÃO PARA A PARTE DO CRUD.

8- PELO CMD OU CODE, ACESSE A PASTA 'SERVER' E EXECUTE O COMANDO 'nodemon .\index.js', 'node .\index.js' ou 'npm rum devStart';

9- PELO CMD OU CODE, ACESSE A PASTA 'CLIENT' E EXECUTE O COMANDO 'npm start';

10- AGORA O PROJETO DEVERÁ FUNCIONAR CORRETAMENTE.
